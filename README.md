# Clawi Trading App

Android app for accessing LN Markets trading platform.

## Features
- WebView wrapper for LN Markets
- Optimized for mobile trading
- Clean, minimal interface

## Download
APK builds are available in the [Releases](../../releases) section.

## Build Locally
```bash
./gradlew assembleDebug
```

APK will be at: `app/build/outputs/apk/debug/app-debug.apk`
